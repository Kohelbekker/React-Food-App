# Food Order App

This is Food Order App build with ReactJS.

List of used React Hooks and other features:

- `useState`
- `useReducer`
- `useEffect`
- `useContext`
- `useRef`
- `React.forwardRef`
- css modules

### How to start App

To start App on the localhost just simply run:

```shell
> npm install

> npm start
```
